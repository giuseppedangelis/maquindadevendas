# Máquina de Vendas

Landing page e páginas institucionais do app `Máquina de Vendas`.

## Requisitos

- Node.js 18+
- npm

## Como rodar

```sh
npm install
npm run dev
```

## Scripts

```sh
npm run dev
npm run build
npm run preview
npm run lint
npm run test
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vitest
