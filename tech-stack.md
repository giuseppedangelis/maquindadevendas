  "build": "vite build",           // Build para produção
  "build:dev": "vite build --mode development", // Build para desenvolvimento
  "lint": "eslint .",              // Executa linting
  "preview": "vite preview",       // Preview do build
  "test": "vitest run",            // Executa testes
  "test:watch": "vitest"           // Executa testes em modo watch
}

## ⚙️ Configurações Especiais

### TypeScript
- **Configuração flexível** com `noImplicitAny: false`
- **Path alias** `@/*` apontando para `./src/*`
- **Modulo ES** habilitado (`"type": "module"`)

### Vite
- **Porta**: 8080
- **Host**: `::` (localhost + network)
- **HMR overlay** desabilitado
- **Path alias** configurado para `@`

### Tailwind CSS
- **Dark mode** baseado em classes
- **Cores customizadas** com sistema HSL
- **Fonte Montserrat** configurada
- **Animações customizadas** definidas
